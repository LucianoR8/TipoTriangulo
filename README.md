# Tipo de Triângulo

## Desenvolvimento de um programa que informa o tipo de triângulo com as medidas fornecidas pelo usuário

---

### Tecnologias utilizadas:

 - Visual Studio;
 - C# .NET 8.0

### Triângulos disponíveis:
 - Retângulo;
 - Obtusângulo;
 - Acutângulo;
 - Isosceles;
 - Equilatero.
